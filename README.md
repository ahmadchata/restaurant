# Restaurant Page

> A bare-bones restaurant website using vanilla JS and Webpack.


## Built With
- JavaScript
- HTML 5 and CSS 3.
- Webpack 5


## Getting Started

- `git clone git@github.com:ahmadchata/restaurant.git`
- `cd restaurant`
- `npm i`
- `webpack`
- Open index.html from dist folder
